    EndpointHandler: (data: any, headers?: MsgHdrs) => Promise<any>

    Service endpoint handler signature. Must return a Promise, resolved value will be included in the response message's data field. Throwing an error from an endpoint handler will respond to the message with an ErrorResponse.
